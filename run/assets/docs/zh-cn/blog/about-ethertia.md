Ethertia 名称起源  
(中文名 "以太效应", "以太的愿望", "以太咏叹调")  
名称由 Ether, aria 的变种组成。希望有神秘 广阔 深奥 忧郁的感觉。  
意为 通过假说中以太这种神秘广阔的介质，找到通往另一个世界的方法  
Ethertia 有些像 Ethernet. 或 Celestia.

没有使用 Etheria 因为它太常见。
